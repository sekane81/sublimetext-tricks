# Sublime text tricks
##Select##

<b>Command + D</b><br> Select a word.
 
<b>Command + L</b><br> 
Select a line.
 
<b>Command + A </b><br>
Select the entire content within the document.
 
<b>Ctrl + Command + M</b><br> 
Select anything inside the bracket (which is useful when working with CSS or JavaScript)
